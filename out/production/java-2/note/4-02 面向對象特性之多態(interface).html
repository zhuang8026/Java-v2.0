1. 什麼是 "多態" ? 淺顯易懂：調用同一個動作，產生不同的行為 官方解釋：多態是同一個行為具有多個不同表現形式或型態的能力 -> 就如同打印機 -> 行為：打印 -> 多個不同表現形式或型態：打印出的效果，可以彩色也能黑白 2. 實現多態的關鍵是 接口 （interface） -> 1. interface 是一個抽象的類型，只提供方法定義 -> 2. "實現類" 是實現特定接口(interface)的類，成為 實現類 -> "實現類" 就是實現 特定接口 的類 -> 一個接口（interface）可擁有多個 "實現類" 3. EX: 1. 創建interface文件選擇 interface -> 內容：public interface xxxxx{ .... } -> 注意：作為接口(interface)，只能定義方法的結構 2. 子類 繼承了 父類的時候，子類必須加 implements -> ............. 子類 ... 實現 .. 父類 { ... } -> public class Sample01 implements Interface { ... } 3. Interface outSideDemo = sv01.contact(87); // 調用 contact() 並傳入不同的值，會產生不同的內容 4. 總結： 1. interface 定義一個接口，對所有的子類（實現類）進行管理和約束 2. 所有的實現類都必須 實現 interface 所定義的方法 ( Interface.java 中的 voice() ) 3. 外部類要使用 實現類，在實例化前面必須是 接口的名字 ( Interface xxx = new Sample01(); )
