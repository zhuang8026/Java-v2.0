1. 包的作用： -> 管理 java 文件 -> 解決同名文件衝突 2. 定義包： package 包名 -> 必須放在 java 源程序的第一行 包名間可以使用 " . " 號隔開(層次) 例子（eg）：com.william.MyClass -> 區分類的名稱： -> com.imooc.william.MyClass -> com.imooc.margerita.MyClass -> 如何使用 package ？ -> 1. 使用 import 關鍵字 : import com.imooc.william.MyClass 3. 重點： -> java中，包的命名規範是全部小寫字母拼音，然後java文件首字母大寫 -> eg: demo_imooc_入門二.* => 加載全部文件 -> eg: demo_imooc_入門二.demo.* => 加載某個具體子包下的指定文件中的全部文件 -> eg: demo_imooc_入門二.demo.Demo6_2_1 => 加載某個具體子包下的指定文件
