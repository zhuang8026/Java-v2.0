1. 成员变量 -> 在類中定義，用來描述對象有什麼 -> ** 定義在 自身類 中 -> ** 可以被 本類 的方法使用，和本類有關係的類使用 -> 2. 局部变量 -> 一般都會定義在 類 的方法中，在方法中臨時保存數據 -> ** 定義在 方法 的內部 -> ** 只能在 當前的方法中使用 -> 3. 什麼是作用域 -> 變量的作用範圍 4. 成员变量 和 局部变量 區別 -> 作用域不同 a. 局部變量 的作用域 僅限於定義他的方法使用（方法的開始 到 方法的結束） b. 成員變量 的作用域 是在整個類都可以使用 （類的開始 到 類的結束） -> 初始值不同 a. java 會給成員變量一個初始值 b. java 不會給局部變量賦予初始值 -> 在同一方法中，不能有重名的局部變量 -> 在同一類中，當成員變量 和 局部變量 重名時，局部變量 具有更高的優先級 （方法中的 局部變量 優先）
